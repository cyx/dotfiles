Dotfiles FTW
============

    cd ~/
    git clone git://github.com/cyx/dotfiles.git .dotfiles
    ~/.dotfiles/bin/symlink-dotfiles

    # You're done!

Assupmtions
-----------

- That you already have a `bash_profile` which `source`s
  your `.bashrc`.
- You're using vim (not basic vi).
- Your default shell is bash
